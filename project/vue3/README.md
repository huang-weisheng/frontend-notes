# vue3 项目说明

## 项目结构
- src - 源代码目录
  - assets - 静态资源
  - components - 组件
  - router - 路由配置
  - stores - Pinia 状态管理
  - views - 页面视图
  - App.vue - 根组件
  - main.ts - 入口文件

## 技术栈
- Vue 3 - 前端框架
- TypeScript - 类型支持
- Vite - 构建工具
- Pinia - 状态管理
- Vue Router - 路由管理

## 开发说明
1. 使用 Volta 管理 Node.js 和 npm 版本
2. 使用 TypeScript 进行开发,提供类型检查
3. 使用 Vite 作为构建工具,提供快速的开发体验
4. 使用 Pinia 进行状态管理
5. 使用 Vue Router 进行路由管理

## 构建部署
- 开发环境: `npm run dev`
- 生产构建: `npm run build`
- 构建产物位于 dist 目录
